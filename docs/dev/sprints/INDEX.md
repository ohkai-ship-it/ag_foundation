# Sprints — Index

This folder contains sprint planning and tracking for ag_foundation.

## Structure
- **SPRINT_LOG.md** — central log for all sprints (single file, no per-sprint folders)
- **SPRINT_PLAN_SPRINT01.md** — detailed plan for Sprint 01
- **SPRINT_REPORT_SPRINT01.md** — Sprint 01 completion report
- **PROCESS.md** — sprint cadence and process guidelines
- `templates/` — sprint plan/report templates (for reference)

## How We Work
1. All sprint tracking goes in **SPRINT_LOG.md** — one section per sprint.
2. Sprint plans (if detailed) are standalone files: `SPRINT_PLAN_<name>.md`.
3. We do **not** create per-sprint folders to avoid sprawl.
4. Link sprint goals to backlog items (AF-000x).
