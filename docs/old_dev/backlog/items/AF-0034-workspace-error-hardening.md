# AF-0034 --- Workspace Error Message Hardening (No Name Leakage)

# Version: v1.1

## Status

Ready

## Priority

P0

## Problem

Workspace errors reveal identifiers and lack actionable guidance.

## Goal

Standardize errors without leaking workspace names and always include
guidance.

## Acceptance Criteria

-   No identifier leakage
-   Clear next steps
-   Tests cover error paths
